+ pip install: pyvisa, libusb1, openpyexcel, matplotlib, numpy
+ Драйверы: NI-VISA full
+ Методика поверки Е14: 4221-008-42885515 МП
+ Методика поверки Е-502: ДЛИЖ.411618.0080 МП
+ Калибратор: Fluke 5500A(5522A)
